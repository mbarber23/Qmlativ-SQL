Hello! This repo stores sql query files of Hamilton Southeastern Schools' Student Information System. The SIS is Skyward Qmlativ; queries are written in Microsoft SMSS and copied into PowerBI desktop.

For access to the Power BI Semantic Model which has various merges and ammends as well as a complex relationship structure, contact dataanalysts@hse.k12.in.us.

We also have several dashboards built that we can share that are built off of this model including District Data, Safety and Attendance dashboards.

To use these dashboards, you will need access to your Skyward Qmlativ database via SQL Read. You can get access to this by opening a ticket (this can be done for free). You will also need access to Power BI; a free version will allow you to do basic data analysis BUT you will need PRO or Premium licenses if you want to share dashboards with others.
